== Description ==

Plugin allows you to create unlimited number of different channels to put in order your flow of leads

== Installation ==

1. Install plugin [WPForms](https://wordpress.org/plugins/wpforms-lite/)
2. Install plugin [Notify.Events](https://wordpress.org/plugins/notify-events/)
3. Install this plugin
4. Activate plugins through the **Plugins** screen (**Plugins > Installed Plugins**)
5. Sign up on website [Notify.events](https://notify.events)
6. Create a channel
7. Add source "WordPress" and acquire token
8. Paste token in **Channels** tab (**Settings** -> **Notify.Events**)
9. Subscribe in your channel to platform you choose
